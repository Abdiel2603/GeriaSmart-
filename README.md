# GeriaSmart - Sistema Integral para el Cuidado de Adultos Mayores

## Descripción del Proyecto

GeriaSmart es una plataforma web y móvil desarrollada para optimizar la gestión, seguimiento y atención de adultos mayores mediante un sistema integral que conecta pacientes, cuidadores y administradores en un solo entorno digital.

El objetivo principal del sistema es mejorar el control de información médica, administración de medicamentos, historial clínico, citas médicas, vinculación entre pacientes y cuidadores, así como la gestión de productos de apoyo orientados al bienestar de los adultos mayores.

El proyecto surge como una solución tecnológica ante la necesidad de centralizar procesos que comúnmente se realizan de forma manual o desorganizada, permitiendo una administración más eficiente, segura y accesible tanto para familiares como para responsables del cuidado.

Además, cuenta con una aplicación móvil complementaria orientada al seguimiento rápido y acceso remoto a la información más importante del paciente.

---

## Problemática

En México, una gran parte del seguimiento de salud de los adultos mayores se realiza mediante métodos informales como libretas, notas físicas o registros no estructurados, lo que provoca:

- Olvido en la administración de medicamentos
- Falta de control sobre citas médicas
- Pérdida de historial clínico importante
- Dificultad para el seguimiento por parte de familiares
- Errores en la toma de decisiones médicas
- Falta de comunicación entre cuidadores y responsables

GeriaSmart busca resolver esta problemática mediante la digitalización y centralización de la información médica y operativa.
